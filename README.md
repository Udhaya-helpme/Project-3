# Project-3
Internship Application Tracker — Project 3

The Internship Application Tracker is a database system that helps college students manage their internship applications. It tracks the full lifecycle of an application from submission to a final hiring decision, including companies, job postings, interview rounds, contacts, and tags.

This project extends the tracker with Redis as an in-memory key-value store. It caches application records to reduce database lookups and tracks each user's recently viewed applications.
